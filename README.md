# USC — Universal Structured Container

**Version:** 0.1 (Foundational Reference Implementation)
**Language:** Python
**License:** MIT

## Overview
USC (Universal Structured Container) is a **text-surfaced, symbol-driven container system**. It allows embedding, referencing, and resolving any form of data—code, media, structured data—inside a human-readable text document.

## Project Structure
- `SPEC.md`: The authoritative specification for USC v0.1.
- `usc/`: Core Python library (Model, Parser, Resolver).
- `cli.py`: Command Line Interface tool.
- `examples/`: Reference `.usc` files.
